# READ THIS FIRST ✅

Welcome! This pack helps you practice **conditionals** and **loops** in JavaScript.

## How to run
1) Open `Practice_Activities/index.html` in Chrome.  
2) Right-click → **Inspect** → open the **Console** tab.  
3) In `index.html`, uncomment ONE `<script>` tag for the Tier you want to try, then refresh.
